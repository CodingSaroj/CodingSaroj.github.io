Ractor

The combination of speed, simplicity and flexibility

Ractor is a 2D and 3D cross-platform game engine designed with speed, simplicity and flexibility in mind. 'Flexibility' here means that it's easy to use interface won't take away it's ability to create complex games. It has features like visual coding and visual shaders which make it very easy to use for beginners. Whoever you are, an absolute beginner, a beginner, an intermediate developer or a professional game developer, this game engine will fulfill most of your needs. Checkout the learn page for tutorials on how to use Ractor.

Features

  • 2D and 3D support
  • Visual Coding
  • Visual Shaders
  • Entity Component System
  • Cross-platform
